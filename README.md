# 🪲 Insecta – Modern Insect Landing Page

A modern and visually attractive insect showcase landing page built using HTML5 and CSS3.

---

## 📖 Project Overview

<b>Live demo:</b> https://koyelbasu.github.io/Insecta/

**Insecta** is a creative front-end landing page project designed with a premium split-screen layout.  
The project focuses on modern UI/UX design, responsive structure, elegant typography, and product-style insect presentation.

This project is perfect for:

- 🎨 Frontend practice
- 💻 Portfolio projects

- 🚀 UI/UX inspiration

---

## 🛠️ Tech Stacks

- 🌐 HTML5
- 🎨 CSS3
- 📦 Flexbox

---

## ✨ Key Features

- 🪲 Premium insect showcase section
- 🔍 Stylish search bar UI
- 🟥 Modern split-screen layout
- 🧾 Insect classification details
- 🛒 Interactive collection button
- 📱 Mobile responsive design
- ⚡ Clean and organized code structure
- 🎯 Modern typography and spacing
